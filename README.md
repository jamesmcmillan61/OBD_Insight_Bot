Developed as part of the University of Hull’s Continuous Development Practice model, this project explores the use of IBM’s Granite model to provide more accessible and meaningful insights into vehicle data.
By uploading a vehicle’s OBD (On‑Board Diagnostics) data, users can interact conversationally with the application to better understand fault codes, vehicle behaviour, and underlying issues. The aim is to transform complex diagnostic data into clear, user‑friendly information through an AI‑driven conversational interface.
